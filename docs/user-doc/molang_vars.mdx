# Molang 变量列表
Ayame 目前支持以下 Molang 变量：

| 变量名称            | 描述             | 类型   |
|---------------------|------------------|--------|
| `aym.has_helmet`      | 玩家是否穿戴头盔 | 布尔值 |
| `aym.has_chest_plate` | 玩家是否穿戴胸甲 | 布尔值 |
| `aym.has_leggings`    | 玩家是否穿戴护腿 | 布尔值 |
| `aym.has_boots`       | 玩家是否穿戴靴子 | 布尔值 |
