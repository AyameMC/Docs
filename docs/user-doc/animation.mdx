### 移动动画

| 动画名称          | 描述                              | 循环播放 |
|-------------------|-----------------------------------|----------|
| move.walk         | 玩家行走时的循环动画。             | 是       |
| move.run          | 玩家疾跑时的循环动画。             | 是       |
| move.climb_still  | 玩家在活板门下方静止时的循环动画。 | 是       |
| move.climbing     | 玩家在活板门下方爬行时的循环动画。 | 是       |
| move.sneak_still  | 玩家潜行但静止时的循环动画。       | 是       |
| move.sneaking     | 玩家潜行并移动时的循环动画。       | 是       |
| move.swim         | 玩家游泳时的循环动画。             | 是       |
| move.swim_stand   | 玩家在水中站立式游泳时的循环动画。 | 是       |
| move.jump         | 玩家跳跃时的单次播放动画。         | 否       |
| move.fly          | 玩家在创造模式飞行时的循环动画。   | 是       |
| move.elytra_fly   | 玩家鞘翅飞行时的循环动画。         | 是       |
| move.ladder_up    | 玩家爬梯子向上移动时的循环动画。   | 是       |
| move.ladder_still | 玩家静止在梯子上时的循环动画。     | 是       |
| move.ladder_down  | 玩家爬梯子向下移动时的循环动画。   | 是       |

—

### 状态动画

| 动画名称          | 描述                          | 循环播放 |
|-——————|——————————|-———|
| state.boat`      | 玩家坐在船上时的循环动画。      | 是       |
| state.sleep     | 玩家睡觉时的循环动画。          | 是       |
| state.ride      | 玩家骑马（或驴）时的循环动画。   | 是       |
| state.ride_pig  | 玩家骑猪时的循环动画。          | 是       |
| state.sit       | 玩家坐下时的循环动画。          | 是       |
| state.idle      | 玩家静止（无操作）时的循环动画。 | 是       |

—

### 动作动画

| 动画名称             | 描述                              | 循环播放 |
|-———————|———————————|-———|
| action.attacked`    | 玩家被攻击时的单次播放动画。         | 否       |
| action.use_mainhand| 玩家使用主手（右手）时的单次播放动画。 | 否       |
| action.use_offhand | 玩家使用副手（左手）时的单次播放动画。 | 否       |
| action.swing_mainhand| 玩家挥动主手（右手）时的单次播放动画。| 否       |
| action.swing_offhand| 玩家挥动副手（左手）时的单次播放动画。| 否       |

—

### 特殊动画

| 动画名称         | 描述                                | 循环播放 |
|——————|————————————|-———|
| special.riptide`| 玩家使用“激流”附魔三叉戟时的循环动画。  | 是       |
| special.death  | 玩家死亡时的单次播放动画。             | 否       |
