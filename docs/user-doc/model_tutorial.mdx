# 模型创作教程
理解 Ayame 的模型格式对模型创作者至关重要，以下是一个标准的 `metadata.json` 示例：

```json title="metadata.json"
{
  "name": "example_model",
  "authors": [
    "Me!!"
  ],
  "description": "Example Model.",
  "license": "CC0-1.0",
  "links": [
    "https://example.org"
  ],
  "tags": [
    "example-tag"
  ],
  "animations": [
  ],
  "version": "1.0"
}
```
